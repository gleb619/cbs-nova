# Database Documentation

## 1. Configuration

Settings from `application.yml`:

- **DDL Mode**: `validate` (Schema validated against Flyway migrations, not auto-generated by Hibernate).
- **Flyway**: `enabled: true`, `locations: classpath:db/migration`.

## 2. Schema

The settings table structure:

```sql
CREATE TABLE settings (
    id          BIGSERIAL PRIMARY KEY,
    code        VARCHAR(255) NOT NULL UNIQUE,
    value       VARCHAR(255) NOT NULL,
    description VARCHAR(255)
);
```

## 3. Flyway Migrations

Located in `starter/src/main/resources/db/migration/`:

- `V20260101101010__init.sql`: Base initialization.
- `V20260410120000__create_settings.sql`: Table creation.

## 4. Infrastructure

Managed via `docker-compose.yml`:

- **PostgreSQL**: Primary database.
- **Keycloak**: Optional identity provider.

## 5. Integration Testing

Uses Testcontainers for isolated database instances:
`jdbc:tc:postgresql:18.3:///cbsnova`
